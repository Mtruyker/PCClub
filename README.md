# PC Club Admin

WPF-приложение для администрирования компьютерного клуба. В проекте есть экраны для управления компьютерами, клиентами, сессиями, тарифами и статистикой. Данные хранятся в SQLite-базе `pcclub.db` в папке локальных данных пользователя.

## Что было изменено

### Окно сессий

- Добавлен `NullToVisibilityConverter` в `WpfApp8/Infrastructure/NullToVisibilityConverter.cs`.
- В `WpfApp8/Sessions.xaml` кнопка `Завершить` теперь показывается только для активных сессий.
- Если у сессии заполнено `EndTime`, кнопка скрывается через привязку:

```xml
Visibility="{Binding EndTime, Converter={StaticResource NullToVisibilityConverter}}"
```

Это убирает ситуацию, когда у уже закрытой сессии всё ещё висит кнопка завершения.

### Адаптивная разметка экранов

Обновлены основные XAML-экраны:

- `WpfApp8/clients.xaml`
- `WpfApp8/computer.xaml`
- `WpfApp8/Tariffs.xaml`
- `WpfApp8/Sessions.xaml`

Изменения в разметке:

- корневые `StackPanel` заменены на `Grid`;
- верхние панели занимают место по содержимому;
- таблицы занимают всё оставшееся пространство окна;
- убрана фиксированная высота таблиц `Height="400"`;
- колонки `DataGrid` получили `Auto`, `*` и `MinWidth`, чтобы интерфейс лучше подстраивался под содержимое;
- колонки действий расширены, чтобы кнопки `Редактировать`, `Удалить` и `Завершить` не обрезались;
- для таблицы сессий оставлена горизонтальная прокрутка, потому что в ней много колонок.
